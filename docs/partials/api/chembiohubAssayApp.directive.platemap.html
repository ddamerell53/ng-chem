platemap
directive in module chembiohubAssayApp

Description

platemap

Directive to render a 96 or 48 well plate map and allow a form for each well to be populated with data. When a well contains data, it changes colour (to green). Currently selected well has a red border.

Usage

as element:
<platemap
       plate-form="{object}"
       schema-form-holder="{object}"
       cbh="{object}"
       save-plate-function="{function}"
       plate-saved="{boolean}">
</platemap>

Parameters

ParamTypeDetails
plateFormobject

Information about the plate being rendered from a separate form (plate dimensions, name etc)

schemaFormHolderobject

Object containing the angular schema form config for the individual well forms.

cbhobject

The top level cbh object to help with globals and configuration

savePlateFunctionfunction

Pass-through function so that when the user saves the phole plate within the directive, the save functions etc can be called in the parent scope.

plateSavedboolean

Boolean passthrough so the parent scope knows a plate has been saved - can be used for a list update callback if required.
