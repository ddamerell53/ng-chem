renderers
service in module chembiohubAssayApp

Description

renderers

This factory provides a variety of methods for correctly rendering cells and table row headers for our handsontable implementation. Each is referenced by key within the renderers variable. Includes different link renderers for different scenarios (image/file/standard) as well as implementations for triggering modals, safe HTML rendering, bullet lists, field totals etc.

Usage

renderers($timeout, $compile, $state, $rootScope, sfPath, sfSelect);

Parameters

ParamTypeDetails
$timeoutobject

Angular library - timeout handler

$compileobject

Angular library - HTML re-rendering compiler

$stateobject

Access to the current page state (url path, parameters)

$rootScopeobject

The root scope

sfPathobject

splits datalocation - json path split into words

sfSelectobject

sfSelect selects a json path from the current object

Returns

object

data object containing all available renderer objects.
