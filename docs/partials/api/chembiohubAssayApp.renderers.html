Description

renderers

This factory provides a variety of methods for correctly rendering cells and table row headers for our handsontable implementation. Each is referenced by key within the renderers variable. Includes different link renderers for different scenarios (image/file/standard) as well as implementations for triggering modals, safe HTML rendering, bullet lists, field totals etc.
