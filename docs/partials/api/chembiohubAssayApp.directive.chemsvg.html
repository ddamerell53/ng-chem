chemsvg
directive in module chembiohubAssayApp

Description

chemsvg

Renders a chemical structure provided in SMILES format as a 2D SVG format image.

Usage

as element:
<chemsvg
       size="{string}"
       smiles="{string}"
       data-type="{string}"
       base-url="{string}"
       properties="{object}">
</chemsvg>

Parameters

ParamTypeDetails
sizestring

Dimensions of the generated image in pixels.

smilesstring

String representation of chemicasl structure in SMILES format

dataTypestring

Not used

baseUrlstring

If the smiles input is from ChemDraw, store that in the scope

propertiesobject

Contains extra configuration parameters from the app or work out correct SMILES from a supplied SMARTS format structure.
