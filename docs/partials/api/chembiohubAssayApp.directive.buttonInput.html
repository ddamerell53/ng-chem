buttonInput
directive in module chembiohubAssayApp

Description

buttonInput

Builds a two-mode form element - an "edit" mode display which is a regular form element and a "readonly" version which is a disabled text form element displaying the field value. This is then accessible by a "click to copy" mechanism provided by (TODO link)

Usage

as element:
<button-input
       copy-model="{object}"
       disabled
       label="{string}">
</button-input>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
copyModelobject

Used to store data for the read and edit states, acts like a form element

disabledboolean

Flag for displaying this element as a disabled form element (not used)

labelstring

Text title for the field
