searchBreadcrumbs
directive in module chembiohubAssayApp

Description

searchBreadcrumbs

Visual representation of applied search parameters. Since each subform is displayed separately, the app needs to show the user which filters, sorts and searches they've applied. Also simplifies removing one aspect of a search.

Usage

as element:
<search-breadcrumbs
       cbh="{object}">
</search-breadcrumbs>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
cbhobject

The top level cbh object to help with globals and configuration
